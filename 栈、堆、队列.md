# 栈 stack (后进先出) (临时变量一般会存储在栈区)
* 允许插入和删除的一段为栈顶，另一段为栈底
* 处于栈顶位置的数据元素成为栈顶元素
* 基本运算
    * 初始化:构建一个空栈
    * 判断空栈
    * 进栈
    * 出栈
    * 取栈顶
    
# 队列 equeue (先进先出) 
* 循环队列 
* 基本运算
    * 初始化一个空队列:  包括前指针 尾指针
    * 判断空队列:       前指针和尾指针相等
    * 判断满队列:       
        * ( 前指针 + 1 )% 容量  等于 尾指针 **** 这里循环队列用来解决队满的方法是少用一个空间元素 也可以通过数组元素数量比较(数组元素数量 == 容量)
    * 入队              
    * 出队
    * 去队列的头部元素
